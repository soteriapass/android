# pswmgr
Password Manager that uses a client/server architecture to store encrypted passwords remotely

# Dependencies
* gRpc
* Protocol Buffers
* OpenSSL
* Sqlite3 (available in repos)

# Supported platforms

         |  GUI  |  CLI  | Server |
-------- | ----- | ----- | ------ |
Windows  |__YES__|__YES__| __YES__|
Linux    |   NO  |__YES__| __YES__|
Android  |__YES__|   NO  |   NO   |
